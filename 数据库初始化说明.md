# 📝 数据库初始化 - 最后一步

---

## 🚀 执行步骤（2步完成）

### 第1步：获取数据库连接字符串

1. 打开浏览器，访问 **https://vercel.com**
2. 登录并进入 **tradehub** 项目
3. 点击顶部的 **Settings**
4. 左侧点击 **Environment Variables**
5. 找到 **DATABASE_URL**，点击显示值
6. 复制整个连接字符串（类似这样）：
   ```
   postgresql://user:password@ep-something.aws.neon.tech/neondb
   ```

---

### 第2步：在终端执行命令

**复制下面的命令到终端：**

```bash
cd /Users/a004/Desktop/binance-trading-dashboard

DATABASE_URL='把第1步复制的连接字符串粘贴在这里' npx prisma db push
```

**示例：**
```bash
DATABASE_URL='postgresql://user:pass@ep-xxx.aws.neon.tech/neondb' npx prisma db push
```

**注意：**
- 要把 `把第1步复制的连接字符串粘贴在这里` 替换成真实的连接字符串
- 保留单引号 `'...'`
- 确保连接字符串没有多余的空格

## ⚠️ 如果找不到DATABASE_URL

说明您还没有在Vercel配置数据库连接：

3. 复制连接字符串
